Simulating the world

Published: June 15, 2025

Agentic IDEs are great for basic crud app development, but can I really trust them to build complex numercial analysis software? Where one incorrect pointer dereference will loose one in a sea of llm written spagetti wasting hours debugging.

So I want to see how far I can get with building a Reinforcement learning based software system to find optimal parking strategies to optimize tarmac operations at an airport. Who knew moving airplanes around was so complicated? Long story short - tugs need to move airplanes out of the way of other airplanes when one randomly calls the tower to land. This usually happens for private jets - lifestyles of the rich and famour amirite?

I'm going to break this problem down into the following steps:

  1. Model the tarmac at the airport.
  2. Create an interactive game to play the tug and visualize the dynamics (side-quest)
  3. Generate random schedules and attempt RL loop to learn the tug's optimal policy

Simple enough.

So far we have a simple world model and interactive game where I can be the tug and move planes around. Repo link coming soon.
